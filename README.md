# To-Do List Application

A clean, modern, and responsive web-based To-Do List application built to help users manage their daily tasks efficiently. 

## Features
* Modern, centered interface container with smooth drop shadows.
* Responsive design that looks great on mobile, tablet, and desktop screens.
* Interactive task management including creating, editing, and deleting items.
* Local storage persistence so your tasks remain saved even after refreshing the page.

## Technologies Used
* HTML5 for structural markup
* CSS3 for custom modern styling and layouts
* JavaScript for dynamic DOM manipulation and data persistence
